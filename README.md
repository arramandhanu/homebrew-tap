# Homebrew Tap

Personal Homebrew tap for my tools.

## Installation

```bash
brew tap arramandhanu/tap
brew install <formula>
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| shipctl | Professional Docker deployment automation tool |

## Adding New Tools

1. Create `Formula/<tool-name>.rb`
2. Push to this repo
3. Users can install: `brew install arramandhanu/tap/<tool-name>`
